
# 输入框(textfield)
textfield包括一个表示键盘输入的水平线，是标准输入框的增强版，拥有更生动的视觉效果。
##如何使用
1、创建含有“u-text”样式的div元素，用于包括整个输入框的内容。

	<div class="u-text">
       
    </div>
2、在div元素中创建样式为“u-input”的输入框
	
	 <div class="u-text">
        <input class="u-input"/>
    </div>

3、在div元素中继续添加样式为“u-label”的label元素，其内容为输入框的提示信息。

	<div class="u-text">
        <input class="u-input"/>
        <label class="u-label">this is lable</label>
    </div>
