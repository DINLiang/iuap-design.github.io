# 月份选择(month)
点击输入框时，会弹出月份卡片，选择的月份会显示在输入框中。
##如何使用
1、定义样式为“u-month”的div元素。
	
	<div class='u-month'>
        
    </div>
2、在第一步创建的div元素中添加样式为“u-input”的输入框。用户选择的时间将会存放在这里。

	<div class='u-month'>
        <input class="u-input" type="text">
    </div>
