模态框 modal

modal控件

区分与message,显示更多的信息,且自带确认取消回调。

插件依赖

依赖于 http://design.yyuap.com/static/uui/3.0.6/js/u.js

用法

1.定义触发弹出事件的DOM

<button id="modalBtn" class="u-button" >Success</button>

2.js 定义应用范围

u.compMgr.apply({
        el:'body'
})

3.获取dom

var msgBtn2 = document.body.querySelector("#msgDialogBtn2");

4.绑定modal事件

u.on(msgBtn2,'click', function(){
        u.confirmDialog({
            msg: "是否保存单据？",       //modal内容
            title: "测试确认",           //modal title
            onOk: function () {          //确认后的回调
                alert('ok')
            },
            onCancel: function () {         //取消后的回调
                alert('cancel')
            }
        });
})

示例

消息摸态框

通过confirmDialog方法调出模态框 当点击取消或者保存，可在js中的方法里进行业务的回调

<button id="msgDialogBtn2" class="u-button raised accent">消息确认框</button>
u.compMgr.apply({
        el:'body'
})
var msgBtn2 = document.body.querySelector("#msgDialogBtn2");
u.on(msgBtn2,'click', function(){
        u.confirmDialog({
            msg: "是否保存单据？",
            title: "测试确认",
            onOk: function () {
                alert('ok')
            },
            onCancel: function () {
                alert('cancel')
            }
        });
})

提示摸态框

只有一个确认按钮 通过messageDialog方法调出模态框

<button id="msgDialogBtn" class="u-button raised accent">消息提示框</button>

   
u.compMgr.apply({
    el:'body'
})
var msgBtn = document.body.querySelector("#msgDialogBtn");
u.on(msgBtn, 'click', function() {
    //            u.showMessage("HELLO!!!");
    u.messageDialog({ msg: "HELLO!!!", title: "测试提示", btnText: "OK!" });
})

自定义摸态框

通过dialog方法调出模态框 用户自定在html文件自定义所要展现的内容 点击保存，做进一步的确认校验

API

js方法

名称 方法参数 回调方法 描述
confirmDialog 1.msg 2.title 1.onOk 2.onCancel 确认和取消都有回调
messageDialog 1.msg 2.title 3.btnText 无 只有确认按钮，自定义确认按钮字样btnText
dialog 1.id 2.content 3.hasCloseMenu 自定义回调 id定义dialog唯一性,content自定义内容,可以是html.确认取消时间自行绑定
